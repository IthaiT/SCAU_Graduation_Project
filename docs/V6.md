| 任务类型   | 回归 (预测收盘价)            | 三分类 (跌 0 / 平 1 / 涨 2)              |

## CEEMDAN 原理简述

CEEMDAN (Complete Ensemble Empirical Mode Decomposition with Adaptive Noise) 将非平稳
金融时间序列分解为多个本征模态函数 (IMF)：
- 高频 IMF → 短期市场噪声
- 低频 IMF + 残差 → 中长期趋势

模型输入包含原始特征 + 各 IMF 分量，获得多尺度时频信息，
显著降低信噪比，使分类器更容易捕捉涨跌趋势信号。

## 标签生成策略

```
return_t = (close[t+1] - close[t]) / close[t]

if return_t < -0.5%  →  label = 0 (跌)
if return_t > +0.5%  →  label = 2 (涨)
else                  →  label = 1 (平)
```
